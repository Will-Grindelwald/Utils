# bitlocker 相关 bat 脚本

## 使用命令解锁 用 bitlocker 加密的驱动器

* ulc.bat 解锁驱动器，脚本中的 N 为加密驱动器的盘符，XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX 为驱动器用 bitlocker 加密时生成的恢复秘钥
* lc.bat 锁定驱动器，脚本中的 N 为加密驱动器的盘符
